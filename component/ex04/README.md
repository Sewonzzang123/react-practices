1. state
   - 컴포넌트 내부의 상태 관리 데이터
   - 상태의 변경은 UI를 다시 렌더링 하게 한다.
   - 클래스 컴포넌트에서는 constructor에서 default 값 세팅으로 생성한다.
   - 함수 컴포넌트에서는 useState라는 후크 함수를 사용해서 초기화 한다.
   - 대체적으로 사용자 이벤트로 변경되거나 통신으로 변경된다.
